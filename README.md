## ⚡ Quick Start

1. **Navigate to the project:**
   ```bash
   cd bet_copilot/
   ```

2. **Initialize database:**
   ```bash
   cd database && python create_db.py
   ```

3. **Start frontend:**
   ```bash
   cd frontend && npm install && npm run dev
   ```

4. **Demo the backend:**
   ```bash
   python demo_backend.py
   ```
